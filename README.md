# My Portfolio — Full Stack DevOps Project

A production-grade portfolio built with microservices, Next.js, PostgreSQL, MongoDB, Socket.io, and AI automation.

## Tech Stack
- Frontend: Next.js 14 + PWA
- Auth Service: Node.js + PostgreSQL
- Content Service: Node.js + MongoDB
- AI Service: Python FastAPI + OpenAI
- Infrastructure: Docker + Kubernetes + GitHub Actions

## Getting Started
cp .env.example .env
npm install
npm run dev
